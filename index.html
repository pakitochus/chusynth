ChuSynth

A Raspberry Pi interface for Fluidsynth.

What is ChuSynth?

Chusynth is a piece of software aimed at developing a powerful, easy-to-use sample player on a Raspberry Pi. It will use FluidSynth as a player, pyFluidSynth as a wrapper and an UI that will show up in PiFace, all of it written in Python.
